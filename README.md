# Customer Behavior and Revenue Analysis

## Project Overview
This project analyzes customer behavior and revenue patterns using **Power BI**. The goal is to understand how customer engagement, purchasing behavior, and demographic factors influence overall revenue and customer lifetime value. The project includes interactive dashboards, visualizations, and key metrics to support data-driven decision making.

## Dataset
The dataset contains customer demographic, behavioral, and transactional information. Key variables include:  
- Age, Gender, Country, City.  
- Membership Years, Login Frequency, Session Duration.  
- Total Purchases, Average Order Value, Lifetime Value.  
- Discount Usage, Return Rate, Churn Status.  

## Tools & Technology
- **Power BI** – Data visualization and dashboard creation.  
- **DAX** – Calculating key metrics like Total Revenue, Avg Order Value, Lifetime Value, and Churn Rate.  
- **Power Query** – Data cleaning and preprocessing.  

## Features
- Analyze total revenue, average order value, and customer lifetime value.  
- Visualize revenue distribution by country, age, and signup quarter.  
- Track customer engagement and retention trends.  
- Identify high-value customers and potential churn risks.  
- Explore payment methods and discount usage patterns.  

## Results
- Strong overall revenue with consistent average order value.
- Highest revenue from the USA and UK, with India as a growing market.
- Longer membership and higher engagement drive higher lifetime value and purchases.
- Payment options and discount usage influence customer buying behavior.
- Churn insights suggest targeted retention and personalized campaigns can improve loyalty.

## Future Scope
- Include additional data sources like social media or browsing history.  
- Apply machine learning to predict churn and lifetime value.  
- Expand analysis to more cities or countries.  
- Enhance dashboards with real-time monitoring.
